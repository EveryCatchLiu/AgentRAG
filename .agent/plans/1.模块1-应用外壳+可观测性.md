## 快速开始

1. **配置环境变量：**
   - 复制 `backend/.env.example` → `backend/.env` 并填写真实值
   - 复制 `frontend/.env.example` → `frontend/.env` 并填写 Supabase 值
2. **运行数据库迁移：** 在 Supabase SQL 编辑器中运行 `backend/migrations/001_threads.sql`
3. **启动后端：** `cd backend && conda activate AgentRAG && uvicorn src.main:app --reload`
4. **启动前端：** `cd frontend && npm run dev`
5. 打开 `http://localhost:5173`
